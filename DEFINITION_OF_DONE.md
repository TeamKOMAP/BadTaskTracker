# Definition of Done (DoD)

Каждая задача считается завершённой, когда выполнены **все** следующие пункты:

## Критерии завершения

### 1. Код написан и соответствует требованиям User Story
- [ ] Реализована вся функциональность, описанная в User Story
- [ ] Выполнены все Acceptance Criteria

### 2. Код прошёл code review
- [ ] Код проверен минимум одним другим членом команды
- [ ] Все замечания по ревью исправлены

### 3. Написаны unit-тесты (где применимо)
- [ ] Для новой функциональности написаны unit-тесты
- [ ] Тесты покрывают основные сценарии и edge cases

### 4. Тесты успешно выполняются
- [ ] Все unit-тесты проходят (зелёный статус)
- [ ] Интеграционные тесты (если есть) также проходят

### 5. Обновлена документация
- [ ] Обновлены комментарии к API (XML-комментарии)
- [ ] При необходимости обновлен README.md
- [ ] Swagger UI корректно отображает изменения

### 6. Функция протестирована вручную
- [ ] QA провёл ручное тестирование
- [ ] Нет критических багов (blockers)
- [ ] Функция работает согласно Acceptance Criteria

### 7. Изменения слиты в main ветку через pull request
- [ ] Создан pull request в ветку main
- [ ] Все пункты DoD выполнены и отмечены в PR
- [ ] PR принят и код слит
